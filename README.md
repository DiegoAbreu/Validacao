# validacao
Pacote para validação de dados
***
Instalação pacote:
```
$ pip install validacao
```
Importação:
```
import validacao
```
***
### Funções disponíveis:
- Validação de CPF.

    Função: validacao.cpf(num_documento)

    Os retornos são True para válido e False para inválido.
```
validacao.cpf('99999999999)
OUTPUT: False
```
